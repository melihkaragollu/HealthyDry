# HealthyDry

## Proje Hakkında



**Proje Tanımı:** 


Bu proje, Süleyman Demirel Üniversitesi Bilgisayar Mühendisliği bölümü Yazılım Mühendisliği dersi ödevi kapsamında geliştirilmektedir. Proje, dondurularak kurutulmuş meyvelerin satışının yapıldığı, veritabanı etkileşimli bir web sitesi ve mobil uygulamadan oluşmaktadır.
